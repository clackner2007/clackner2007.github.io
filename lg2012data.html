Data from Lackner & Gunn, 2012

The tables of model parameters for the bulge-disk fits to SDSS galaxies from Lackner and Gunn, 2012 are below. These are ascii tables. There is one table for each galaxy model and each table contains all the galaxies in the sample (71,825). Descriptions of the columns in each table can be found in the header of each file.

If you use this data, please cite Lackner & Gunn, 2012. If you are interested in other outputs, such as the covariances between model parameters, please contact me.

File Model Description
lackner2012_galmodels_nb4.dat exponential disk + de Vaucouleurs bulge models (classical bulge model)
lackner2012_galmodels_nb1.dat exponential disk + exponential bulge models (pseudo-bulge model)
lackner2012_galmodels_exp.dat exponential profile models (bulgeless disk model)
lackner2012_galmodels_dvc.dat de Vaucouleurs profile models (elliptical galaxy model)
lackner2012_galmodels_ser.dat Sersic profile models
