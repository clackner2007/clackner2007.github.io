Data from Lackner & Gunn, 2012

Tables for bulge-disk decomposition from Lackner and Gunn, 2012. These are plain text tables. There is one table for each galaxy model, and each table contains all the galaxies in the sample (71,825). The model chosen in the paper for each galaxy is denoted by a three character code the TYPE column in the tables.

lackner2012_galmodels_nb4.dat exponential disk + de Vaucouleurs bulge models (classical bulge model)
lackner2012_galmodels_nb1.dat exponential disk + exponential bulge models (pseudo-bulge model)
lackner2012_galmodels_exp.dat exponential profile models (bulgeless disk model)
lackner2012_galmodels_dvc.dat de Vaucouleurs profile models (elliptical galaxy model)
lackner2012_galmodels_ser.dat Sersic profile models

If you use this data, please cite Lackner & Gunn, 2012. If you are interested in other outputs, such as the covariances, please email me.
